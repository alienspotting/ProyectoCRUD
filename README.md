# Proyecto CRUD - Gestión de Clientes

Aplicación de escritorio desarrollada en C# WinForms con acceso a SQL Server.

El proyecto implementa un sistema CRUD completo (creación, lectura, actualización y eliminación de registros) basado en una arquitectura de acceso a datos mediante procedimientos almacenados en SQL Server.

Este proyecto surge durante mis prácticas de 1º de DAW como iniciativa personal. Me basé en la forma de trabajar de la empresa donde realicé las prácticas para entender mejor cómo funciona su ERP y para conseguir aplicar esos conceptos en una aplicación de gestión propia.

Un proyecto muy interesante, puesto que me permitió integrar los conocimientos de diferentes asignaturas del ciclo en un único proyecto por primera vez, utilizando técnicas similares a las usadas en un entorno profesional real.

## Funcionalidades
- Listado de clientes
- Alta de clientes
- Edición de clientes
- Eliminación de clientes
- Filtros por empresa, nombre y estado operativo

## Tecnologías
- C#
- WinForms
- SQL Server
- ADO.NET

## Base de datos

El proyecto incluye scripts SQL para la creación de la base de datos, así como las tablas, inserción de datos de prueba, y los procedimientos almacenados necesarios para el uso de la aplicación.
